== Description ==

This plugin allows you to embed Brainshark presentations into your post using the shortcode [brainshark url="YOUR_PRESENTATION_URL" height="452" width="555"].
Optimal embed sizes included 345x292, 440x366, 555x452

== Installation ==

Installation is easy:

   Download the plugin.
   Unzip the plugin file.
   Upload the brainshark folder to the plugins directory of your blog.
   Enable the plugin in your admin panel.
   Start embedding Brainshark presentations!
